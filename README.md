# composer
Minimalistic GUI key composer

# dependencies
- [meson](https://mesonbuild.com)
- [libgtk-3-dev](https://www.gtk.org)
- [libxdo-dev](https://github.com/jordansissel/xdotool)

# install and run
```shell
meson build
cd build
ninja && ./composer
```
